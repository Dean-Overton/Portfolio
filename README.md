### Local Testing
1) ```yarn start``` to run the app in the development mode.

To create a local server (test on different devices):

1) CLI: ```yarn build``` to build the project.
2) ```cd build``` and ```http-server --push-state```
3) Open [http://10.7.18.10:8080](http://10.7.18.10:8080) to view it in the browser on phone.

### Production Build
